Qu'est-ce que le
HTML ?

HTML, ou HyperText Markup Language, est le langage
utilisé pour créer des pages web. Pensez-y comme
la structure d'une maison : il définit où vont
les murs, les fenêtres et les portes.

Structure de Base

Une page HTML se compose de plusieurs parties essentielles :

  • <!DOCTYPE html> : Indique que le document est un fichier HTML.
  • <html> : L'élément racine du document.
  • <head> : Contient des informations sur la page (titre, styles, etc.).
  • <body> : Contient tout le contenu visible de la page.

Éléments HTML

Les éléments HTML sont les composants de base.
Voici quelques exemples :

  • <h1> à <h6> : Titres de différentes tailles.
  • <p> : Paragraphes de texte.
  • <a> : Liens vers d'autres pages ou sites.
  • <img> : Images insérées dans la page.

Attributs HTML

Les attributs donnent des informations supplémentaires
sur un élément. Par exemple :

  • href : Utilisé dans les liens pour spécifier l'URL.
  • src : Utilisé pour indiquer la source d'une image.
  • alt : Fournit une description textuelle d'une image.

Exemples Pratiques

Voici un exemple simple de code HTML :

                    
    <!DOCTYPE html>
    <html lang="fr">
        <head>
            <title>Ma Première Page</title>
        </head>
        <body>
            <h1>Bonjour, monde !</h1>
            <p>Ceci est un exemple de page HTML.</p>
        </body>
    </html>

Dans cet exemple, nous créons une page qui affiche "Bonjour, monde !" en tant que titre et un petit texte en dessous.
