Qu'est-ce que PHP ?

PHP (Hypertext Preprocessor) est un langage de script côté serveur, largement utilisé pour le développement web. Il permet de créer des pages dynamiques en générant du contenu HTML en temps réel.

Syntaxe

La syntaxe de PHP est assez simple. Un script PHP commence par et le code PHP est écrit entre ces balises. Voici un exemple : < ?php echo "Bonjour, le monde !"; ?>

Variables

Les variables en PHP commencent par le signe dollar $. Voici quelques exemples :

  • $nom = "Alice";
  • $age = 25;
  • <a> : Liens vers d'autres pages ou sites.
  • $pays = "France";

Fonctions

Les fonctions en PHP sont définies à l'aide du mot-clé function. Voici un exemple de fonction simple : function saluer() { return "Bonjour !"; }

Exemples Pratiques

Voici un exemple d'utilisation de PHP pour afficher la date actuelle :

                <?php
                  date_default_timezone_set('Europe/Paris'); // Définit le fuseau horaire
                  $heure = date('H:i:s'); // Récupère l'heure au format heures:minutes:secondes
                  echo "L'heure actuelle est : " . $heure;
                ?>

Pour voir cet exemple en action, rendez-vous à l'URL http://localhost/heure.php.
