Qu'est-ce que
JavaScript ?

JavaScript est un langage de programmation dynamique,
principalement utilisé pour créer des pages web interactives.
Il permet de manipuler le contenu HTML et CSS pour offrir
une expérience utilisateur améliorée

Syntaxe

La syntaxe JavaScript est basée sur des instructions, des expressions et des déclarations. Voici un exemple de déclaration d'une variable : let message = "Bonjour, le monde !";

Variables

Les variables sont utilisées pour stocker des valeurs. En JavaScript, vous pouvez déclarer une variable en utilisant var, let ou const. Voici des exemples : var nom = "Alice"; let age = 25; const pays = "France";

Fonctions

Les fonctions sont des blocs de code qui peuvent être appelés à tout moment. Voici un exemple de fonction simple : function saluer() { alert("Bonjour !"); }
