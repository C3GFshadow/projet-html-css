Qu'est-ce que CSS ?

CSS (Cascading Style Sheets) est un langage de style utilisé pour
décrire l'apparence d'un document écrit en HTML. Il permet
de contrôler la mise en page, les couleurs, les polices
et bien plus encore.

Syntaxe CSS

La syntaxe de base du CSS se compose de sélecteurs et de déclarations. Une déclaration comprend une propriété et une valeur. selector { property: value; }

Sélecteurs CSS

Les sélecteurs sont utilisés pour cibler les éléments HTML. Voici quelques types de sélecteurs :

  • Sélecteur de type : p (cible tous les paragraphes)
  • Sélecteur de classe : .ma-classe (cible les éléments avec la classe "ma-classe")
  • <a> : Liens vers d'autres pages ou sites.
  • Sélecteur d'ID : #mon-id (cible l'élément avec l'ID "mon-id")

Propriétés CSS

Il existe de nombreuses propriétés CSS. Voici quelques-unes des plus courantes :

  • color : définit la couleur du texte.
  • background-color : définit la couleur de fond.
  • font-size : définit la taille de la police.
  • margin : définit l'espace autour d'un élément.

Exemples Pratiques

Voici un exemple simple de code HTML :

                  
                    #pourquoi{
                        display: flex;
                        margin-top: 18vh;
                    }
                    #pourquoi h2{
                        color: black;
                    }
                    #pourquoi .l-pourquoi{
                        margin-left: 600px;
                    }
                    #pourquoi .img-pourquoi-web{
                        height: 90%;
                    }
                    #pourquoi .r-pourquoi{
                        margin-left: 380px;
                        text-align: left;
                        display: flex;
                        flex-direction: column;
                    }
